EpicHyperSketch.CountMinSketch — Type
CountMinSketch(motif_size::Integer; delta, epsilon, use_cuda)

Create a CountMinSketch object.

  • motif_size: Number of features (motifs).
  • delta: Error probability.
  • epsilon: Error tolerance.
  • use_cuda: If true, use CUDA arrays.

EpicHyperSketch.Record — Type
- vecRefArray::Vector{AbstractArray{IntType, 3}}
Reference arrays on the selected features for each data point

- combs::AbstractArray{IntType, 2}
Combinations, store each combination horizontally. For example, 
this can be generated by 
```julia
using Combinatorics
combinations(1:5, 3) |> collect |> hcat
````
